## Configuration
Features :

**FR :** 
* Création de Job
* Création d'un point boss
* Création d'un coffre
* Suppression de Job avec reboot
* Suppression de Job via un nom !
* La langue (FR/EN)

**EN :**
* Job creation
* Creation of a point leader
* Creation of a chest
* Removing Job with reboot
* Job removal via name!
* Language (FR/EN)

## Requirements
* [es-extended](https://github.com/Vanheden/es_extended)

## Download
* [Download](https://github.com/Leap0p0/popo_job/archive/refs/heads/main.zip)
* Put it in your server resources folder.

## Installation
Add ``ensure popo_job`` in the **server.cfg** file.
